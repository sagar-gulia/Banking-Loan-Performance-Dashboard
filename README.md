# Banking-Loan-Performance-Dashboard

Short Discription /Purpose
Banking Loan Performance Dashboard is an interactive data visualization project that provides a comprehensive overview of loan operations and portfolio health. It highlights key metrics such as total loan amount, approval rate, default rate, net exposure, regional loan distribution, and branch performance.

The dashboard enables stakeholders to:
Monitor loan trends over time
Compare approval vs. rejection rates
Analyze loan distribution across regions
Identify top-performing branches by loan volume
Track overall portfolio risk and exposure

Tech Stack

Microsoft Excel – Data source, preprocessing, and initial analysis
Python – Core programming language
NumPy – Numerical computations
Matplotlib & Seaborn – Data visualization
Jupyter Notebook – Development and analysis environment
Power BI – Interactive dashboard creation and reporting

📊 Data Source

The data used in this project is sourced from a banking loan dataset stored in Microsoft Excel format. It includes information on loan applications, approval status, loan amounts, regional distribution, branch performance, and default indicators.
This dataset is used for analysis, visualization, and building an interactive dashboard to monitor loan performance and portfolio health.
